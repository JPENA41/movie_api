Instructions Page

The web application will provide users with access to information about various movies, directors, and genres. Users can sign up, update their personal information, and build a list of their favorite movies.

Request URL Responses
GET /movies Return a list of every movie.
GET /movies/:title Return a particular movie by title.
GET /genres/:genre Return data about a specific movie genre.
GET /directors/:directorsName Return data about a director by name.
POST /register Allow registation.
PUT /users/:id/:infoToUpdate/:newValue Allow user to update their username.
POST /users/:id/:favorites/:newFavorite Allow user to add a movie to the list of favorites.
DELETE /users/:id/unregister Unregister a existing user from the database.
DELETE /users/:id/favorites/:deleteFavorite Remove a favorite movie from the user's favorites.
